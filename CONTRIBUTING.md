Thank you for your interest! Bug reports are welcomed.

## How to Report a Bug
1. Check the existing GitHub Issues to see if the bug was already reported.
2. If not, open a new issue.
3. Include your Python version, OS, clear steps to reproduce, and the error logs.

## How to Request a Feature
1. Open a new issue.
2. Label it as a `feature request`.
3. Explain your use case and why this feature is valuable to users.
